# 歷史程式與長文（`legacy-archive` 分支）

自 **v0.9.4** 起，**`main` 預設分支不再包含 `archive/` 目錄**，以免 GitHub 首頁根目錄出現大量歷史資料夾（`accelerators/`、`paper/`、`configs/` 等）。

完整舊樹仍保留在 Git 分支 **`legacy-archive`**（與移除前之 `main` 快照一致）。

- **瀏覽目錄**：[github.com/.../tree/legacy-archive/archive](https://github.com/dragonheart8787/AI-For-prediction/tree/legacy-archive/archive)
- **切換本機分支**：`git fetch origin legacy-archive && git checkout legacy-archive`

歷史 `requirements*.txt` 匯總仍在該分支的 `archive/requirements_legacy/`。
