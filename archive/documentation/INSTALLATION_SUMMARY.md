## 🔧 故障排除

### 如果安裝失敗
1. **檢查網絡**: 確保網絡連接正常
2. **更新 pip**: `python -m pip install --upgrade pip`
3. **使用鏡像**: `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple <package>`
4. **手動安裝**: 根據錯誤信息手動安裝失敗的包

### 如果測試失敗
1. **重新安裝**: 刪除已安裝的包，重新安裝
2. **檢查版本**: 確保 Python 版本 >= 3.8
3. **使用虛擬環境**: 創建新的虛擬環境
4. **查看日誌**: 檢查安裝報告了解詳細信息
